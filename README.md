# Motif UI Showcase

A polished marketing/documentation concept site built with **Next.js** and **Tailwind CSS** for **Motif UI**, the React component library created within **TÜRKSAT's e-Devlet** ecosystem.

## Included

- Next.js App Router structure
- Tailwind CSS v4 setup
- Clean, light visual language inspired by modern component-library docs
- Interactive playground section
- Real gallery cards using the provided Motif UI screenshots
- GitHub + Storybook entry points

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Notes

- Assets from the conversation are placed in `public/` and `public/showcase/`.
- The page is intentionally built as a single high-impact landing/documentation experience so it can be expanded into a fuller docs site later.
